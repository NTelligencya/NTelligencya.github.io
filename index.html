01 Power Query: Surveillance Data Analysis Available

Learn Microsoft Excel's Power Query through six exercises set inside a fictional dystopian Australian outback town. You'll combine messy CSV files, merge relational tables, write custom M-language functions, and reshape data for executive briefings. The dataset doubles as a discussion prompt about algorithmic bias and surveillance ethics.

DURATION · 3 hrs LEVEL · Beginner to intermediate FORMAT · HTML dossier + 13 CSVs
Open course
// what you will learn
  • Loading and cleaning CSV data
  • Combining files from a folder
  • Merge queries and aggregations
  • Pivot and unpivot transformations
  • Custom functions in the M language
  • Cross-tabulation and group-by analysis
